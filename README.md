# Laboratorio 6
Scene Management & Asynchronous Transitions

## Requisitos

### 1. 3 niveles jugables
Se implementaron 3 niveles jugables con la misma condición de victoria, pero distinto nivel de dificultad.

### 2. Menú principal
Se tiene un Main menu con opción de una nueva partida o reanudar partida.

### 3. Managers persistentes
Los managers persistentes realizados fueron **MainMenuManager**, **PersistenceManager**, **AudioManager** y **GameManager**.

### 4. Avance de nivel
Los niveles se avanzan al llegar con la princesa en cada uno, es decir, cumplir con la condición de vistoria.

### 5. Pantalla de Carga/Loading Screen
Se implementó una loading screen asincrónica que muestra una barra de progreso y el número de progreso que se lleva.
